Introduction

Drawl is a drawing language. That means that it allows you to represent drawings as programs, then render the programs into images.

Examples

As an example, the following code:

public class Drawl {
public static void main(String[] args) {
System.out.println("Hello World");
Drawing drawing = new Drawing();
Circle circle1 = new Circle();
drawing.add(circle1);
Circle circle2 = new Circle();
drawing.add(circle2);
circle2.setRightOf(circle1);
String svg = drawing.getSVG(200, 100);
System.out.println(svg);
}
}

This simple example illustrates the basic idea, but it can be extended to create many beautiful and useful drawings.
