◆ DEFAULT

com.aarrelaakso.drawl.LineEnding.Type.DEFAULT

The default line ending. Synonyms: NORMAL, TRIANGLE.

For the default line ending, we set the constraint of having the legs have twice the length of the base.

The width of this triangle can be calculated as:

\[w^2 + (\frac{h}{2})^2 = (2h)^2\]

\[w^2 + \frac{h^2}{4} = 4h^2\]

\[w^2 = 4h^2 - \frac{h^2}{4}\]

\[w^2 = \frac{16h^2}{4} - \frac{h^2}{4}\]

\[w^2 = \frac{15h^2}{4}\]

\[w = \sqrt{\frac{15h^2}{4}}\]

\[w = \frac{h \sqrt{15}}{2}\]

To calculate the area of the triangle:

\[Area = \frac{hw}{2}\]

Given that we want the area to be 16:

\[ \frac{hw}{2} = 16\]

\[hw = 32\]

Substituting in our formula for w

\[h \frac{h \sqrt{15}}{2} = 32\]

\[\frac{h^2 \sqrt{15}}{2} = 32\]

\[h^2 \sqrt{15} = 64\]

\[15 h^4 = 4096\]

\[h^4 = \frac{4096}{15}\]

\[h = \sqrt[4]{\frac{4096}{15}}\]

\[h \approx 4.07\]

Then, to calculate the width of the triangle:

\[\frac{wh}{2} = 16\]

\[wh = 32\]

\[w = \frac{32}{h}\]

\[w \approx 7.87\]

◆ DIAMOND

com.aarrelaakso.drawl.LineEnding.Type.DIAMOND

A diamond-shaped line ending. Synonyms: RHOMBUS

We set the height of the rhombus to be equal to its sides:

\[h = x\]

\[Area = \frac{w \times h}{2}\]

Solving for width in terms of height:

\[h^2 = (\frac{h}{2})^2 + (\frac{w}{2})^2\]

\[(\frac{w}{2})^2 = h^2 - (\frac{h}{2})^2\]

\[\frac{w^2}{4} = h^2 - \frac{h^2}{4}\]

\[w^2 = 4h^2 - h^2\]

\[w^2 = 3h^2\]

\[w = h \sqrt{3}\]

Substituting into the formula for area:

\[16 = \frac{h^2 \sqrt{3}}{2}\]

\[32 = h^2 \sqrt{3}\]

\[h^2 = \frac{32}{\sqrt{3}}\]

\[h = \frac{\sqrt{32}}{\sqrt[4]{3}}\]

\[h \approx 4.3\]

Now calculating w:

\[w = \sqrt{3} \frac{\sqrt{32}}{\sqrt[4]{3}}\]

\[w \approx 7.44\]

◆ DOT

com.aarrelaakso.drawl.LineEnding.Type.DOT

A dot-shaped line ending. Synonyms: DISK

For the dot/disk line ending, we need

\[\pi r^2 = 16\]

so

\[r^2 = 16/\pi\]

\[r = \sqrt{\frac{16}{\pi}}\]

\[r = \frac{4}{\sqrt \pi}\]

\[r \approx 2.26\]
